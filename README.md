# StandupBot

Telegram bot that automates asynchronous daily standups: private DM prompts, single nudge, consolidated digests to team channels, and searchable archived history.

Spec: [`docs/blueprint.md`](docs/blueprint.md).

Built on [agnt-gm.ai](https://agnt-gm.ai). The whole bot is built and refined here as pull requests across successive build passes.
